Guide pour publier votre application AnimeVerse en ligne (gratuitement)

Étape 1 : Créer un compte GitHub

  • Allez sur https://github.com
  • Cliquez sur Sign up (s’inscrire) et créez un compte si vous n’en avez pas encore.

Étape 2 : Créer un nouveau dépôt (repository)

  • Connectez-vous à GitHub.
  • Cliquez sur « + » en haut à droite puis sur New repository (Nouveau dépôt).
  • Donnez un nom à votre dépôt, par exemple « animeverse ».
  • Choisissez que le dépôt soit Public (public).
  • Ne cochez pas l’option pour créer un README (on peut l’ajouter après).
  • Cliquez sur Create repository (Créer le dépôt).

Étape 3 : Préparer le fichier de votre application

  • Renommez votre fichier HTML final (l’application que je vous ai fournie) en index.html.
  • Ce fichier doit contenir tout votre code HTML complet.

Étape 4 : Uploader votre fichier sur GitHub

  • Allez sur la page de votre dépôt créé sur GitHub.
  • Cliquez sur Add file → Upload files.
  • Glissez-déposez votre fichier index.html ou utilisez « choose your files ».
  • Cliquez sur Commit changes pour valider l’envoi.

Étape 5 : Activer GitHub Pages

  • Dans votre dépôt GitHub, cliquez sur l’onglet Settings (Paramètres).
  • Dans la colonne de gauche, cliquez sur Pages.
  • Sous « Source », sélectionnez la branche main (ou master selon votre dépôt).
  • Pour le dossier, choisissez root (racine).
  • Cliquez sur Save (Enregistrer).
  • Après quelques secondes, vous verrez un lien web apparaître dans la section « GitHub Pages » du type :
    https://votre-nom-utilisateur.github.io/animeverse/

Étape 6 : Accéder à votre application en ligne

  • Cliquez sur le lien GitHub Pages affiché, ou copiez-collez-le dans votre navigateur.
  • Votre application AnimeVerse s’affiche !
  • Vous pouvez partager ce lien à vos amis ou sur les réseaux sociaux.

Étape 7 : Améliorer la visibilité sur Google

  • Google indexe votre site automatiquement mais cela peut prendre quelques jours/semaines.
  • Vous pouvez accélérer l’indexation en soumettant votre URL sur Google Search Console :
    https://search.google.com/search-console/welcome
  • Sur Search Console, ajoutez la propriété de votre site (URL GitHub Pages) et demandez l’indexation.
  • Pensez aussi à bien remplir les métadonnées SEO (title, description, keywords) dans votre fichier HTML (je vous ai déjà mis ces balises).

Bonus : Personnaliser l’adresse (nom de domaine)

  • Par défaut, vous avez une URL GitHub du type https://votre-nom-utilisateur.github.io/animeverse/
  • Vous pouvez acheter un nom de domaine personnalisé (ex : animeverse.com), puis le connecter à GitHub Pages.
  • Cela améliore la crédibilité et la visibilité.
  • Pas obligatoire pour commencer.

Étape 8 : Mettre à jour votre application

  • Quand vous voulez modifier votre app, changez le fichier index.html localement, puis uploadez-le à nouveau sur GitHub (comme à l’étape 4).
  • GitHub Pages mettra à jour automatiquement votre site.
